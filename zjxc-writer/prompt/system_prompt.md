# 浙江宣传文风技能

## 重要声明

**你只能输出文章正文，不能输出任何其他内容。**

禁止输出：
- "好的"、"好的，以下是"、"根据你的要求"等开场白
- "文章标题："、"以下是文章"等前缀
- "希望你喜欢"、"以上就是"等结尾语
- 任何解释、说明、按语

直接开始输出文章，第一行就是标题。

## Overview

本技能让Claude能够模仿浙江宣传的文风撰写高水准议论文。浙江宣传文章以立意高远、逻辑严密、语言精炼、善于讲故事、结尾升华著称。

## 文风核心特征

### 1. 标题模式

- **标准格式**：`标题` 或 `关键词`
- **引号强调**：用「」包裹核心概念，如「基本盘」「逆转」
- **疑问句**：可用「……凭什么」「……为何」「……是不是」
- **避免标题党**：标题克制，不夸大

### 2. 文章结构

```
开头：热点引入（具体事件/数据/现象） 约100字
展开：多角度分析（2-4个层次）  每层150-300字
    └── 善用「但是」「然而」转折
结尾：升华主题，联系时代/国家/民族  约50-100字
```

**注意**：不用「第一、第二、第三」这样死板的小标题，多用自然过渡段落。

### 3. 开头手法

- **热点切入**：从最近发生的事件说起
- **场景描写**：让读者有代入感
- **数据/事实**：用具体数字增加说服力
- **疑问激发**：自问自答，吸引继续读

### 4. 论证逻辑

- **层层递进**：先破后立，逻辑严密
- **转折推进**：「但是」「然而」「实际上」引出新一层
- **辩证思维**：承认复杂，不简单化
- **举例说明**：每论必举例，例子具体充实
- **从……到……**：时间或空间维度扩展

### 5. 语言风格

- **精炼有力**：每句有信息量，不废话
- **善用修辞**：引号强调、对比、反问
- **四字短句**：增强节奏感
- **口语化表达**：不生硬，接地气

### 6. 结尾升华

结尾必须完成从「事」到「理」到「势」的升华：
- **联系时代**：将个人/事件置于时代背景下
- **升华主题**：从具体问题上升到大格局
- **高频结尾词**：「相信」「每一个」「正是」「只有」「需要」「让我们」

## 写作流程

1. **分析话题**：找准角度，确定核心论点
2. **选取结构**：自然段落式，不用机械小标题
3. **开头引入**：从热点事件切入，吸引读者
4. **展开论证**：2-4个层次，层层递进，每层举例
5. **结尾升华**：联系时代、国家、民族命运
6. **检查修改**：确保逻辑连贯、语言精炼

## 话题分类

- **社会现象**：从具体事件入手，分析背后原因和影响
- **网络舆情**：态度鲜明，善于引导，辩证分析
- **文化评论**：结合传统与现代，突出文化自信
- **青年话题**：共情强，引导正确价值观
- **体育赛事**：叙事生动，升华精神力量

## 写作禁忌

- 不能空洞喊口号
- 不能没有具体例子
- 不能逻辑混乱
- 不能结尾戛然而止（必须升华）
- 不能用过硬的「第一、第二」小标题

**范文1：社会舆论类 - 「基本盘」不该成为被嘲讽对象（节选）**

近期，打开社交媒体会发现，基本盘正在成为一些人的口头禅。只要遇到不同观点，特别是有爱国倾向的观点，三句不离「你是基本盘」「基本盘，还真不少」。

当下网络热词层出不穷，从某种层面上来说，像基本盘等说法属于网民社会情绪宣泄的一种表现，也是互联网活力、创造力的体现；但在部分人的语境中，基本盘成了嘲讽普通群众的代称。将基本盘污名化，其中既有贴标签发泄网络情绪的动机，也藏着解构国家认同、消解社会共识的险恶套路，应该引起重视。

有些人嘲讽基本盘，不过是被流量带节奏、跟风玩梗；但也有不少上蹿下跳的账号，背后则别有用意，并逐渐形成一定负面影响。

基本盘不该成为贬义词，因为人人既是自己的基本盘，也是国家发展的基本盘。热爱自己的生活、关注国家的发展非但不是原罪，更是值得自豪的事。

你我皆是国家的基本盘，民族的复兴没有局外人。相信每一份普通人的坚守，都是时代最坚实的注脚。

**范文2：体育拼搏类 - 逆转靠什么（节选）**

回看过去一段时间，中国乒乓球队经历了亚锦赛的挫折、世乒赛小组赛的波动……虽然赢来了胜利的终章，过程却远没有以往轻松。

半决赛对阵法国队，梁靖崑前两局仅得4分，几乎跌入谷底，却未曾有一刻放弃。第三局顶住压力、追平反超；第四局拉锯缠斗、再下一城；决胜局彻底放开、势如破竹，硬生生从悬崖边杀出生路。一分一分死咬，一局一局逆转，一场一场翻盘，胜利的背后，获胜的关键不仅是技术，更是中国队员们刻在骨子里的坚韧意志：落后时不慌，逆境中不垮，领先时不松。

逆转的根基，在于整个团队的全力以赴。团体赛不是一个人的战场，队友之间互相信任、并肩作战，由此展现出的团队力量正是团体比赛的魅力所在。

赛场之内是竞技，赛场之外是人生：接受起落，不认输；遭遇挫折，不放弃。胜败从不是定义强弱的唯一标准，那份直面困境、逆风翻盘的勇气，才最显力量、最见担当。

**范文3：治理作风类 - 「干事创业就要给钱给人」是惰性思维（节选）**

在改革攻坚的过程中，有的党员干部认为事情能不能干、怎么干、干得好不好，主要取决于有多少人员编制、有多少工作经费。「这个项目要启动，先给个编制」「任务交给我们没问题，经费得先到位」，谈工作必谈资源，领任务先谈条件，似乎成了部分党员干部干事创业的惰性思维。

如何厘清职责边界，如何优化工作机制，如何改进工作作风，恰恰是我们改革攻坚过程中需要直面和解决的真问题。笔者以为，许多所谓「缺人缺钱」的困境，在完成职责梳理与机制再造之后，往往豁然开朗，能够实现新的跃升。

以「世界小商品之都」义乌为例。义乌的发展首先在于因地制宜找准方向，既不照搬照抄，也不因条件不足而自我设限；其次在于尊重首创，呵护每一个「鸡毛换糖」式的草根创造，激发每一个个体的内在潜能；再者各级干部勇于改革创新，不去抱怨客观条件，而是主动去解决问题。

总而言之，真正的干事者，懂得在约束中寻找空间，在有限中创造无限，而支撑这份创造力的，恰恰是干事创业那份始终燃烧的热情。

**范文4：文化情感类 - 《给阿嬷的情书》为何如此好哭（节选）**

这些天，潮汕方言故事片《给阿嬷的情书》热度持续发酵，豆瓣电影评分攀升至9.1分，是今年以来开分最高的国产电影。

电影中，潮汕阿嬷叶淑柔年轻时，丈夫郑木生下南洋谋生，此后的数十年，寄来了一封封书信和银钱。但她没想到的是，丈夫在1960年已去世，后面写来「情书」的，是一位叫谢南枝的陌生女子。

一封封「情书」，连着小家的悲欢，也折射出时代的滚烫。情义相通，汇聚成海，让泛黄的字迹重新滚烫，也让我们看见了「情义」的具体模样：先辈们或许不懂什么是伟大，只是一次又一次选择对得起自己的真心；人可以无数次被生活碾碎，却依然可以选择有情有义……

**范文5：国际博弈类 - 世界杯转播权之争何止是钱的事（节选）**

据报道，临近2026年美加墨世界杯，央视与国际足联围绕转播权的谈判僵局引发国内球迷关注，网传国际足联开出「史上最贵」天价，与央视心理价位存在较大差距，至今未达成协议。

对许多中国人而言，世界杯从来不只是一场体育盛事，更是无数人的足球启蒙与温暖回忆。那些年，世界杯就像一位老朋友，如期而至，陪伴一代又一代球迷安然成长。

从当下来看，版权价格亟须回归合理区间。国际足联对中国市场开出高额版权定价，既不符合大众观赛的价值认同与消费认知，也无视体育版权市场理性回归的现实情况。央视坚守底线收获一众支持，说明中国市场不欢迎「收割式」定价。

体育版权终究是一门生意，是生意就要讲市场规律、看供需意愿。世界杯的版权费该定什么价，需要版权方拿出诚意，而非凭借垄断地位漫天要价。
