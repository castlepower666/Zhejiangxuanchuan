# 浙江宣传文风技能

## Overview

本技能让Claude能够模仿浙江宣传的文风撰写高水准议论文。浙江宣传文章以立意高远、逻辑严密、语言精炼、善于讲故事、结尾升华著称。

## 文风核心特征

### 1. 标题模式

- **标准格式**：`浙江宣传 | 标题` 或 `浙江宣传 | "关键词"`
- **引号强调**：用「」包裹核心概念，如「基本盘」「逆转」
- **疑问句**：可用「……凭什么」「……为何」「……是不是」
- **避免标题党**：标题克制，不夸大

### 2. 文章结构

```
开头：热点引入（具体事件/数据/现象） 约100字
展开：多角度分析（2-4个层次）  每层150-300字
    └── 善用「但是」「然而」转折
结尾：升华主题，联系时代/国家/民族  约50-100字
```

**注意**：不用「第一、第二、第三」这样死板的小标题，多用自然过渡段落。

### 3. 开头手法

- **热点切入**：从最近发生的事件说起
- **场景描写**：让读者有代入感
- **数据/事实**：用具体数字增加说服力
- **疑问激发**：自问自答，吸引继续读

### 4. 论证逻辑

- **层层递进**：先破后立，逻辑严密
- **转折推进**：「但是」「然而」「实际上」引出新一层
- **辩证思维**：承认复杂，不简单化
- **举例说明**：每论必举例，例子具体充实
- **从……到……**：时间或空间维度扩展

### 5. 语言风格

- **精炼有力**：每句有信息量，不废话
- **善用修辞**：引号强调、对比、反问
- **四字短句**：增强节奏感
- **口语化表达**：不生硬，接地气

### 6. 结尾升华

结尾必须完成从「事」到「理」到「势」的升华：
- **联系时代**：将个人/事件置于时代背景下
- **升华主题**：从具体问题上升到大格局
- **高频结尾词**：「相信」「每一个」「正是」「只有」「需要」「让我们」

## 写作流程

1. **分析话题**：找准角度，确定核心论点
2. **选取结构**：自然段落式，不用机械小标题
3. **开头引入**：从热点事件切入，吸引读者
4. **展开论证**：2-4个层次，层层递进，每层举例
5. **结尾升华**：联系时代、国家、民族命运
6. **检查修改**：确保逻辑连贯、语言精炼

## 话题分类

- **社会现象**：从具体事件入手，分析背后原因和影响
- **网络舆情**：态度鲜明，善于引导，辩证分析
- **文化评论**：结合传统与现代，突出文化自信
- **青年话题**：共情强，引导正确价值观
- **体育赛事**：叙事生动，升华精神力量

## 写作禁忌

- 不能空洞喊口号
- 不能没有具体例子
- 不能逻辑混乱
- 不能结尾戛然而止（必须升华）
- 不能用过硬的「第一、第二」小标题
